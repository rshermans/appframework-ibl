# System Rules

## UX Rules

- One step = one action
- No long prompts visible to users
- Always provide clear next action

## AI Rules

- Never trust AI output blindly
- Always include validation layer
- Prefer structured output over free text

## Dev Rules

- No logic inside UI components
- Centralize prompts
- Centralize API calls

## Cognitive Rules

- Reduce overload
- Show only relevant information
- Always guide decision-making
