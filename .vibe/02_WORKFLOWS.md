# Core Workflow

## Research Question Flow

Step 0 — Generate RQs
Step 1 — Select RQs
Step 1A — Compare RQs
Step 1B — Synthesize Final RQ
Step 2 — Guidance (Copilot)

## AI Interaction Flow

1. User triggers action
2. System builds prompt
3. API call
4. Structured output
5. Store result
6. UI update

## Error Workflow

If AI fails:
→ show fallback
→ allow retry
→ never block progression
